# Overview

As a software engineer looking to expand beyond web development, I wanted to explore Rust — a systems programming language known for its memory safety and performance. This project is a starting point to get familiar with the Rust toolchain, project structure, and basic syntax.

This is a simple "Hello, World!" program written in Rust. It uses `cargo` (Rust's build tool and package manager) to compile and run a binary that prints a greeting to the terminal.

The purpose of this project is to set up a simple hello world on a working Rust development environment on Windows and understand the basics of how Rust compiles and runs code. It serves as a foundation for building more complex Rust projects in the future.

# Development Environment

- **IDE:** Visual Studio Code (VS Code) with the rust-analyzer extension
- **Build Tool:** Cargo (included with Rust)
- **Compiler Toolchain:** Rust (installed via [rustup](https://rustup.rs/))
- **Linker:** MSVC Build Tools (Visual Studio Build Tools with the "Desktop development with C++" workload)
- **OS:** Windows 11

The programming language used is **Rust** (2024 edition). Rust is a statically typed, compiled language focused on performance, reliability, and memory safety — all without needing a garbage collector.
